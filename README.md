# defect_detection_example_data
Provides the source images to be used with the example script of the `defect_detection` repository.


### How to use

Get the [defect_detection repository](https://github.com/lovaslin/defect_detection/tree/main) and setup the package
```bash
git clone https://github.com/lovaslin/defect_detection.git
cd defect_detection
pip install -e .
```

From the `example` directory, run the `get_source_images.sh` script
```bash
cd example
./get_source_images.sh
```
The source inmages needed to run the example script should have been automatically downladed from this repository and moved under the right directory.
